# Руководство пользователя

## Начало работы

### Главное окно
При запуске вы видите:
1. **Область перетаскивания файлов** - перетащите файл мышью
2. **Текстовое поле** - для вставки текста
3. **Кнопки управления**:
   - "Выбрать файл" - открыть диалог выбора
   - "ПРОВЕРИТЬ НА ИИ" - запустить анализ
   - "Управление маркерами" - редактировать базу
   - "История" - просмотр прошлых проверок

## Форматы файлов

| Формат | Расширения | Поддержка |
|--------|-----------|-----------|
| PDF | .pdf | ✅ Полная |
| Word | .docx, .doc | ✅ Полная |
| Текст | .txt, .log, .csv, .md | ✅ Полная |
| RTF | .rtf | ✅ Полная |
| ODT | .odt | ✅ Полная |
| HTML | .html, .htm | ✅ Полная |

## Анализ текста

При анализе проверяются:
- Тригонометрические обозначения (tg vs tan)
- Термины линейной алгебры (собственное значение vs eigenvalue)
- Шаблонные фразы
- Канцелярские обороты

## Результаты

- **0-20%** - текст человека
- **21-45%** - есть признаки ИИ
- **46-70%** - высокая вероятность ИИ
- **71-100%** - критический уровень

## Управление маркерами

В редакторе маркеров можно:
- Добавить новый маркер (регулярное выражение, вес, категорию)
- Редактировать существующие
- Удалять ненужные
- Включать/отключать маркеры

## Экспорт отчётов

Доступные форматы:
- HTML - с подсветкой и стилями
- JSON - для машинной обработки
- TXT - простой текстовый отчёт

## Горячие клавиши

- `Ctrl+V` - вставить текст из буфера
- `Ctrl+O` - открыть файл
- `F1` - справка
